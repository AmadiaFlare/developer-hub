An FDC attestation proves information external to the Flare's EVM state.
For example, information can be an asserting that a payment has not been made on a UTXO chain like Bitcoin or Dogecoin, or event logs from an EVM transaction.

An attestation type defines a format of a request for an attestation and a format and rules for the response.
